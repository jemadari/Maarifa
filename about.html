Kuhusu Maarifa Project

Maarifa ni project ya kujifunza ambapo ninalenga kupata ujuzi wa vitendo katika kutengeneza web applications kwa kutumia teknolojia mbalimbali. Project hii ni kama safari yangu ya kujifunza, ambapo natengeneza kurasa, routes, na database connection ili kufahamu jinsi development inavyofanyika.

Malengo

  • Kujifunza kutumia Node.js kama server environment.
  • Kutumia Express kutengeneza routes na API.
  • Kuelewa Postgres kama database ya kuhifadhi taarifa.
  • Kujifunza kuunganisha front-end na back-end kwa pamoja.

Kwa Nini Maarifa?

Jina "Maarifa" limetumika kwa sababu lengo kuu ni kujifunza na kupata ufahamu wa teknolojia za kisasa zinazotumika kwenye development ya web applications. Hii project ni darasa la vitendo kwa ajili yangu na mtu yeyote anayepitia code hizi.
