Karibu kwenye Maarifa

Hapa utajifunza jinsi ya kuunda applications kwa kutumia Node.js, Express, na Postgres. Safari ya maarifa inaanza sasa!

Node.js

Jifunze server-side JavaScript na uunde applications zenye nguvu na scalable.

Express

Pata ufahamu wa framework maarufu ya Node.js kwa ajili ya API na web apps.

Postgres

Tambua jinsi ya kutumia database yenye nguvu, salama, na open-source.
